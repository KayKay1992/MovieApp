## üñºÔ∏è 1. `Image` Component

The `Image` component is used to display a static image in your app.

### ‚úÖ Basic Usage:

```tsx
import { Image } from 'react-native';

<Image
  source={{ uri: 'https://example.com/image.png' }}
  style={{ width: 100, height: 100 }}
/>
```

### ‚úÖ You can also use a **local image**:

```tsx
<Image
  source={require('./assets/logo.png')}
  style={{ width: 100, height: 100 }}
/>
```

### üîß Props:

| Prop         | Type                                                        | Description                                |
| ------------ | ----------------------------------------------------------- | ------------------------------------------ |
| `source`     | `{ uri: string }` or `require()`                            | The image to display (remote or local).    |
| `style`      | `object`                                                    | Defines width, height, border radius, etc. |
| `resizeMode` | `'cover'`, `'contain'`, `'stretch'`, `'center'`, `'repeat'` | How the image should be resized.           |

## üåÑ 2. `ImageBackground` Component

`ImageBackground` lets you set an image **as a background for a container**, so you can overlay text, buttons, or other content on top of it.

### ‚úÖ Basic Usage:

```tsx
import { ImageBackground, Text } from 'react-native';

<ImageBackground
  source={{ uri: 'https://example.com/background.jpg' }}
  style={{ width: '100%', height: 200 }}
>
  <Text style={{ color: '#fff', padding: 10 }}>Overlay Content</Text>
</ImageBackground>
```

### üìå Key Differences from `Image`:

* `ImageBackground` acts like a `View` with an image as the background.
* You can place child components inside it.
* Ideal for hero headers, cards with overlays, etc.

### üîß Props:

| Prop         | Type                             | Description                                       |
| ------------ | -------------------------------- | ------------------------------------------------- |
| `source`     | `{ uri: string }` or `require()` | The background image source.                      |
| `style`      | `object`                         | Controls layout size of the background container. |
| `imageStyle` | `object`                         | Styles specifically for the background image.     |
| `resizeMode` | `'cover'`, `'contain'`, etc.     | How the background image should be resized.       |
